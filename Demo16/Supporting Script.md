Use this when live data will make your agent much much better.

## How to Use the Weather API Key
Step 1: Go to https://www.weatherapi.com/ and Sign Up for free
Step 2: After you have completed the Sign Up, Login
Step 3: After successful Login, you will be redirected to the Dashboard where you will find the API key
Step 4: Copy the API Key and add it to your .env folder.
